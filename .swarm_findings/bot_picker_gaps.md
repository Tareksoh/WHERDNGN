## Notes on Saudi Pro Tactic Coverage

| Tactic | Covered? | Where / Gap |
|---|---|---|
| Trump pull priority (J+9 first, save A-of-trump) | Partial | B-98 lock stops pulling after J+9 gone, but `highestTrump` in bidder branch leads A before 9 unless J is still the highest unplayed — no explicit "play J before A" forced ordering |
| Last-trick +10 targeting | No | Absent from all pickers; MASTER_REPORT B-64 confirmed last-trick bonus IS scored in rollout, but no picker specifically hoards a winning card to claim trick 8 |
| Counting cards in late game (post-trick-4) | Partial | `suitCardsOutstanding` exists; B-96/B-98 use memory post-trick-3; no general "who holds what" inference beyond void flags |
| Sweep (Al-Kaboot) detection 6/0 → push 8/0 | No | matchPointUrgency nudges aggression when leading 7/0 but no picker specifically detects "one Gahwa / Takweesh win ends the match with a sweep bonus" and maximally exploits it |
| Squeeze plays (force opponent to discard a winner) | No | No squeeze detection anywhere in Bot.lua or BotMaster |
| Coup en passant (under-trump to keep boss) | No | pickFollow pos-3 plays lowest trump when only trump winners exist, which approximates this — but is not the same; the deliberate under-ruff of a current leader to preserve a boss for later is not modeled |
| Defensive entry preservation | No | pickLead defender branch leads "low from longest" which implicitly preserves high cards, but no explicit "I need this entry to cash my long suit later" calculation |
| Long-suit establishment ("running") | No | HighestUnplayedRank used to find a current boss; 3-card runs (A gone, K is boss, Qxx follow) are not detected or planned for |
| Belote (K+Q trump) holding patterns | No | K and Q of trump scored individually in `suitStrengthAsTrump`; no special weight for the pair, no "preserve the pair to declare Belote later" reasoning |
| Carre setup recognition | No | J+9 synergy bonus exists in `suitStrengthAsTrump` (+18 Advanced); 4-card honors (J+9+A+T) receives no additional Carre bonus; no Carre-state tracking in play decisions |
